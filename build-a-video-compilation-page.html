Data Analyst Journey

Learn step by step on how to become a data analyst from the foundational skills to then actual handling, of data with programming. Lets Dive into it

Excel / Spreadsheets (Foundation)

In this section you will learn about Excel and spreadsheets for Data analyst as a foundational skill.

What to learn

  • Data entry, formatting, and cleaning.
  • Formulas & functions (SUM, VLOOKUP, IF, INDEX/MATCH).
  • Pivot tables & charts for quick insights.

Why it matters:

  • Almost every company uses Excel.
  • Teaches you the basics of data organization and summarization.
  • Helps you communicate results with simple visuals.

SQL (Data Extraction & Organization)

In this section you will learn about my SQL and its importance to data extraction & Organization.

What to learn:

  • Core commands: SELECT, WHERE, GROUP BY, ORDER BY.
  • Joining multiple tables
  • Aggregations (COUNT, SUM, AVG).

Why it matters:

  • Real-world data is usually stored in databases.
  • SQL lets you pull the exact data you need instead of relying on pre-made spreadsheets.
  • It builds on the logic you learned in Excel but applies it to larger datasets.

Python (Advanced Analysis & Visualization)

In this section you will learn python and its importance to data analysts.

What to learn:

  • Libraries:
  • Pandas → Cleaning & analyzing datasets.
  • Matplotlib/Seaborn → Visualizing data.
  • NumPy → Numerical analysis.
  • Data cleaning (handling missing values, duplicates).
  • Automating repetitive tasks.

Why it matters:

  • Python scales where Excel cannot (millions of rows, automation).
  • Lets you build dashboards, run predictive models, and tell a stronger story with data.
  • It connects Excel-like analysis with database-level scale.
